### <a id="arcadia-ru"></a>🎮 Project Arcadia

Экспериментальный прототип на Unity с процедурными уровнями, лёгким подходом ECS и фокусом на чёткие ощущения от игры. Цель: чистая архитектура и быстрая итерация.

➡️ К проекту Unity: [Labyrinth – Echo of the Shards](Labyrinth-Echo-of-the-Shards/)

### <a id="lyritune-ru"></a>🎵 LyriTune (Text‑to‑Song)

Преобразует короткие тексты в простые мелодии. Первые блоки DSP и небольшой веб‑API в процессе. Демонстрации и исходники будут добавлены по мере готовности.

### <a id="skillforge-ru"></a>📚 SkillForge (платформа обучения)

Платформа с фокусом AE: модульная концепция, система викторин и чистые REST интерфейсы. Сейчас проектирую модель данных и первые эндпоинты. Скоро больше подробностей.

➡️ Репозиторий: [SkillForge – Learn Platform](https://github.com/Velichka81/learn-platform)

### <a id="time-tracking-ru"></a>⏱️ Тайм‑трекер

Инструмент учёта времени с понятным интерфейсом. Цель: быстрый ввод, наглядная аналитика и простой экспорт.

➡️ Репозиторий: [Тайм‑трекер](https://github.com/Velichka81/Zeiterfassung)
